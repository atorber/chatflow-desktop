# ChatFlow Desktop

## 特性

📦 一键启动chatflow  
🎯 支持网页版和PC版WeChat  

![Description](https://github.com/user-attachments/assets/e413ebc5-9427-4255-85dd-52d29b4f7156)

## 快速开始

```sh
# 下载代码
git clone https://github.com/atorber/chatflow-desktop.git

# 进入目录
cd chatflow-desktop

# 安装依赖
npm install

# develop
npm run dev
```
